# WACREN CYBERSECURITY

Building Africa's Cybersecurity Trust Infrastructure for Research and Education — connecting trusted communities, training initiatives, operational incident response teams, and sectoral threat intelligence.

## Features
- **Bot Protection**: Contact information (phone and email) is secured with a "click to reveal" mechanism.
- **Official Branding**: Uses official SVG brand assets for BlueSky, Mastodon, and LinkedIn.
- **Smooth Navigation**: Fluid mobile menu transitions without layout jumps.
- **Scroll-to-Top**: Integrated floating button for easy navigation on long pages.
- **Modern Design**: High-contrast, brand-aligned aesthetics for maximum readability and professionalism.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
